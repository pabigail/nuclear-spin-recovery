# Nuclear Spin Recovery

**Status:** This software is still under development. You are welcome to email **poteshman@uchicago.edu** for support.  

## Overview

This package provides tools for simulating NV center coherence and reconstructing nuclear spin environments from experimental data.  

## Dependencies

To run this software, the following Python packages are required:

- `python` (3.12 recommended)  
- `numpy`  
- `pandas`  
- `pycce`  

## Installation

The recommended way to start is:

```bash
git clone <this-repo-url>
cd nuclear_spin_recovery
pip install .
